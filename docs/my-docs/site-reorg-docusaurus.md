# Docs-as-code site for an object-oriented transactional database

A Docusaurus documentation site covering a set of articles on an object-oriented transactional database platform with a long production history in financial services, government, and healthcare.

Built as a docs-as-code portfolio project: articles researched from primary source documentation, authored in Markdown, and organized by audience and task type. I automatically deploy the site to GitHub Pages through a GitHub Actions workflow designed to grow with additional validation steps over time.
