# Open-source project documentation: VectorLint

Documentation written for VectorLint, an open-source LLM-based prose linter that enforces style guides using an LLM-as-a-Judge approach, as a contribution to the project. The proposed docs set includes:

- Quickstart and installation how-tos
- A tutorial for running VectorLint and iterating on style rules based on findings
- Rule pack authoring for advanced style rule configuration
- CI integration and quality scoring how-tos

These articles were written in coordination with the project lead, using MDX on Mintlify, a common toolchain at developer-tool companies.
